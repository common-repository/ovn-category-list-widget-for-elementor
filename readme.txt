== Installation ==
 
The plugin is simple to install:
 
1. Download ovn-category-list-elementor.zip
1. Upload `ovn-category-list-elementor` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
